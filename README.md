# 🔥 Exprover

IoT 기반의 탐사 로봇으로, 위험 지역을 탐사하기 위해 만들어진 로봇입니다.  
다양한 디바이스(스마트폰, 컴퓨터 등)를 사용하여 원격 조종이 가능하며, 센서를 통해 수집된 데이터를 실시간으로 웹 기반 대시보드를 통해 확인할 수 있습니다.
